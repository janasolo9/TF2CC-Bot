This is the code used for the TF2 Coaching Central (TF2CC) Pug and Moderation Discord bot.
Initially created by Jamillia#4845

Save the BOT_TOKEN in a file called .env in the same folder as main.py

Written in python (Python 3.11)
Uses
- nextcord (pip install nextcord)
- nextcord ext menus (pip install nextcord-ext-menus)
- aiosqlite (pip install aiosqlite)
- dotenv (pip install python-dotenv)
